Nedir Bu Python? Yılan mı?


Python 90’lı yılların başında Amsterdam’da Guido Van Rossum tarafından geliştirilmeye başlanan bir programlama dilidir. Zannedilenin aksine ismini piton yılanından değil, Rossum’un çok sevdiği MonthyPython isimli komedi grubunun adından alır.

MontyPython...

They try it once then go to sleep just when you want to go again...

Tam Olarak Ne İşimize Yarıyor?

Python nesne yönelimli, yorumsal,modüler ve etkileşimli, yüksek seviyeli bir dildir. Programlama dilleri makine mantığı ile insan mantığı arasında köprüdür. Yani makine üzerinde daha hızlı çalışabilir.



---Python ile bir çok işi daha az kod satırı ile halledebiliriz.---



print("Hello World")



Sadeliği ön planda tuttuğu için öğrenmesi diğer dillere göre daha kolay olabilir.

Python Kullanmalı Mıyım?

Evet! Python'u C, C++ gibi dillerden ayıran özelliği derlemeye gerek duymadan çalışabilmesidir. Biz buna yorumsal dil adını veriyoruz. Ayrıca Python içerisinde kullanmamızı bekleyen bir çok fonksiyon var ve bu bizi fonksiyon yazmaktan kurtarıyor.

print() input() range() float() ascii() bytes()

Github Nedir?

Git bir projenin bir çok kişi ile aynı anda yapılabilmesini sağlayan bir platformdur. Github ise projelerimizi sakladığımız depoya denir. Bu depo bir uzak sunucudur ve bu sunucuya erişip kendi projelerimizi atabiliriz. İstersek de başka projelere göz atabiliriz.



Github'ın da içinde kendine ait kodlar (komutlar) olduğunu biliyor muydunuz?

Kısaca Github Komutları

Eğer Github komutlarını araştırmak isterseniz buraya tıklayabilirsiniz.



Şimdi burada Github komutlarının ne olduğunu konuşalım. Github komutları bize projelerimizi hızlı bir şekilde yönetme şansı verir. Bir projeyi gönderme, alma, yorum yazma, yedekleme gibi bir çok fonksiyonu içerisinde bulunduran Github bize kod yazarken büyük rahatlık sağlıyor ve zaman kazandırıyor.
